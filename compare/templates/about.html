Tomify by Phillip Ashe

About

Born from a New England Patriots fan's desire to learn more about Python, facial recognition, and the Django framework; Tomify was created in order to "scratch the itch" I had to explore all three. Facial recognition is powered by the OpenCV library using the "cnn" face detection method, and the site backend is powered by Django 2.1.5. The site is capable of parsing multiple faces, but returns a binary response: either the picture contains Tom Brady, or it does not. Each detected face is labeled as being Tom Brady, or "not the GOAT" (Greatest Of All Time).

Tomify stores submitted images, so that I can continue to train and improve it. Thank you for helping my model "get smarter"!
